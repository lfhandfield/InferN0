#Installation

Prerequisites:
  - gcc
  - Rpackages: Rcpp, RcppArmadillo, ggplot2

Alternatively, use provided script to install prerequisites:

  git clone https://github.com/lfhandfield/InferN0.git
  cd InferN0
  sudo sh install.sh
